Este script foi uma das estratégias de mineração de dados que usei para a minha pesquisa de doutorado. 
As entrevistas foram analisadas por este algoritmo com o intuito de extrair as palavras mais frequentes para subsidiar a construção de nuvens de palavras que embasaram a análise de conteúdo. 


Sobre o Script

Este script em Python recebe um arquivo de texto (entrevista),  remove as pontuações do arquivo. 
Separa em uma lista as palavras com quatro ou mais caracteres. 
Remove as palavras da lista que estão no arquivo de texto Excluir.
Mostra em uma lista ordenado por frequência as 50 palavras que mais se repetem. 
Mostra as 20 palavras mais frequentes. 
Por fim, repete as 20 palavras mais frequentes de acordo com o número de vezes que elas aparecem no texto.
